# F1 AI Analyst Product Ideas

This folder captures product ideas discussed for evolving the project from a head-to-head telemetry tool into a broader F1 race intelligence platform.

Each file describes one idea independently so future coding sessions can reference the exact concept before making implementation changes.

## Idea Files

- `01-race-intelligence-platform.md` - overall product direction.
- `02-structured-controls-and-natural-language-autofill.md` - dropdown-first UX with natural-language shortcut.
- `03-analysis-modes.md` - race overview, strategy, telemetry, head-to-head, beginner mode, and more.
- `04-card-level-ai-explanations.md` - explain individual dashboard cards on demand.
- `05-beginner-and-advanced-insight-levels.md` - explanation depth for new and experienced fans.
- `06-throttlelabs-social-funnel.md` - Instagram/TikTok content loop for acquisition.
- `07-fan-voting-and-driver-rankings.md` - driver voting, race-week rankings, season charts, and share cards.
- `08-fastf1-performance-and-caching.md` - speed strategy for FastF1 loading and derived data.
- `09-ai-cost-and-monetization-controls.md` - controlling LLM cost with metering, caching, and paid tiers.
- `10-monetization-model.md` - free, paid, creator, affiliate, and sponsorship paths.
